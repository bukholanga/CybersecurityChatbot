# Cybersecurity Awareness Chatbot 
 
A C# console chatbot for PROG6221 Part 1 
 
## How to Run 
1. Open in Visual Studio 
2. Press F5 to build and run 
3. Follow the on-screen prompts 
 
## Topics Covered 
- Password Safety 
- Phishing 
- Safe Browsing 
- Malware 
- Email Safety 
- Social Media
